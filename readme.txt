Display a login notice to your website's visitors.

== Description ==

The `Visitor Login Notice` plugin displays a notice to non-logged in user reminding them to login to your website.

== Features ==

* Display a sticky login notice bar (visible even if the user scrolls up / down).
* Choose position on your website where the notice will be displayed (header / footer).
* Customize the notice title & message.
* Add custom login & registration URLs.
* Add a custom button to display on the notice bar, if needed.
* Show / hide the notice from login & registration pages and exclude specific URLs.
* Allow close functionality. When the user closes the notice, the plugin stores a cookie on the user's browser. If the cookie is detected, the notice is not shown again to the user.
* Trigger close event using custom CSS class.
* Customize background and button colors.


== Installation ==

1. Upload the "Visitor Login Notice" plugin into the `wp-content/plugins/` directory or install the .zip via the WP plugins panel in your Wordpress admin dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! You should now be able to access the Plugin's options via `Visitor Login Notice` link under Settings on the WordPress dashboard menu.
