Display a login notice to your website's visitors.

== Description ==

Visitor Login Notice plugin displays a notice to non-logged in user reminding them to login to your website.

== Installation ==

1. Upload "Visitor Login Notice" plugin into the directory `wp-content/plugins/`.
2. Enable "Visitor Login Notice" plugin.
